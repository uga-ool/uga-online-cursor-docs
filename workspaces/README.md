# Workspace examples (optional)

These files are **examples only**, not required team workspaces. Clone paths and which repos you use vary by person.

- **[`examples/`](examples/)** — Sample `UGA-Online-Develop`, `UGA-Online-Reference`, and `UGA-Online-Full` layouts. Copy, rename, and fix `path` entries for your machine.
- **Your own workspace** — Keep a `.code-workspace` wherever you prefer (e.g. next to your repos under `local_dev/`). Cursor does not read workspace files from this repo automatically.

See [`docs/cursor/pr-and-code-review.md`](../docs/cursor/pr-and-code-review.md) for when to use Develop vs Reference vs Full.
