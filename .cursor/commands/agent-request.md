description: Start an eLC AI agent feature request using the designer template
---

You are helping an instructional designer request a new Brightspace agent plugin from a developer.

1. Open and follow [`docs/ai-agents-brightspace/guides/AGENT-REQUEST-TEMPLATE.md`](../../docs/ai-agents-brightspace/guides/AGENT-REQUEST-TEMPLATE.md).
2. If the user has not filled the template, ask structured questions: agent name (kebab-case), purpose, Brightspace inputs needed, tool categories, output format, LLM instructions.
3. Produce a completed brief the user can paste into a new chat in the agent framework repo (or hand to a developer).
4. Remind the user: developers read `AGENT-GUIDE.md` — designers do not need to explain the framework.
5. Do not write plugin code in this command unless the user explicitly switches to implementation in an agent repo.

Optional prompt file: [`docs/cursor/prompts/agent-feature-request.md`](../../docs/cursor/prompts/agent-feature-request.md).
